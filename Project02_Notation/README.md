## Project 2: Infix and Postfix Notation

Write a utility class that converts an infix expression to a postfix expression, a postfix expression to an infix expression and evaluates a postfix expression. 

Infix notation is the notation commonly used in arithmetical and logical formulas and statements. It is characterized by the placement of operators between operands called ”infixed operators”, such as the plus sign in "2 + 2". 

Postfix notation is a notation for writing arithmetic expressions in which the operands appear before their operators. There are no precedence rules to learn, and parentheses are never needed. 

Create a document showing test runs of the program.
A Graphical User Interface (GUI) and JUnit Test is provided. Interface classes are provided.

### Concepts Covered:
- Generic queue
- Generic stack
- Exception handling
