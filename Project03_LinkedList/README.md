## Project 3: Linked List

Write a generic double-linked list class with an iterator, and a generic sorted double-linked list class with an iterator that inherits from your generic double-linked list class.

Create a document showing test runs of the program.
A Graphical User Interface (GUI) and JUnit Test is provided.

### Concepts Covered:
- Exception handling
- Generic Classes
- Double Linked List
- Ordered Double Linked List 
- Iterators 
- Comparators
