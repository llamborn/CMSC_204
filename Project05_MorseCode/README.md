## Project 5: Morse Code Converter

Write the classes required to create a Morse Code Converter Utility. Your Morse Code Converter Utility will be using a generic linked binary tree with generic TreeNodes to convert Morse Code into English. 

Create a document showing test runs of the program.
A Graphical User Interface (GUI) and JUnit Test is provided. Interface classes are provided.

### Concepts Covered:
- Generic Classes
- Utility Class (all static methods)
- Linked Trees
- Building a Tree for conversion purposes
