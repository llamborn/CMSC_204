## Project 4: Course Database

Write a program that creates a database of courses. It will either read from a file of courses or allow the user to add one course at a time. 

Create a document showing test runs of the program.
A Graphical User Interface (GUI) and JUnit Test is provided. Interface classes are provided.

### Concepts Covered:
- Hash table
- Link list
- Hash code
- Buckets/Chaining
- Exception handling
- Read/Write files
